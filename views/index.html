URL Shortener Microservice

Instructions:

  • Use "https://usms.glitch.me/url/" + your link to be shortened to get new link.
  • Do not omit "HTTP://" or "HTTPS://"
  • Omitting will give you an error

Example Input:

    https://usms.glitch.me/url/ + https://google.com ⇒ https://usms.glitch.me/url/https://google.com

Example Output:

    { "Orginal": "https://google.com", "Shortened": "https://usms.glitch.me/2" }
